# NewsHub Jekyll Theme

A professional, SEO-optimized Jekyll theme for news and blog websites.

## Features

- Full SEO optimization (TDK, Open Graph, Twitter Card, Canonical URL)
- Structured Data (JSON-LD)
- Automatic sitemap.xml and robots.txt
- IndexNow GitHub Actions workflow
- Social share buttons
- Article features (TOC, Breadcrumbs, Related Posts, Author Box)
- Performance optimization (Lazy loading, preloaded fonts)
- Responsive design
- Example posts and categories

## Quick Start

1. Install dependencies: `bundle install`
2. Configure `_config.yml`
3. Run locally: `bundle exec jekyll serve`
