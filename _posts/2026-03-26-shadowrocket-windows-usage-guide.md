## 📚 Shadowrocket Windows 使用教程：Clash for Windows 配置指南

许多 Windows 用户都在寻找 **Shadowrocket Windows 使用教程**，希望能像在手机上一样方便地配置代理。由于 Shadowrocket 官方没有 Windows 版本，我们通常会使用功能强大的替代品——**Clash for Windows**。本文将为您提供一份详细的配置指南，手把手教您**小火箭 Windows 怎么配置节点**，以及**Clash for Windows 怎么添加订阅**，助您轻松实现代理上网。

### 💡 Clash for Windows：Shadowrocket 的完美替代

Clash for Windows 是一款基于 Clash 内核的图形界面代理客户端，支持多种代理协议，并且能够完美兼容 Shadowrocket 的订阅链接格式。这意味着，您在其他平台（如 iOS）上使用的“小火箭节点”订阅，可以直接在 Clash for Windows 中使用。

### 第一步：下载与安装 Clash for Windows

如果您尚未安装 Clash for Windows，请参考我们的 [Shadowrocket Windows 怎么下载？小火箭 Windows 版下载教程](/shadowrocket-windows-download-guide/) 获取并安装最新版本。

### 第二步：获取节点订阅链接

在配置 Clash for Windows 之前，您需要一个有效的节点订阅链接。您可以从以下途径获取：

*   **付费机场服务商**：购买机场服务后，服务商会提供专属的订阅链接。
*   **免费节点分享**：在一些社区论坛或 Telegram 频道，会有用户分享免费的订阅链接。请注意其稳定性和安全性。

### 第三步：Clash for Windows 怎么添加订阅 / Shadowrocket 节点怎么导入

这是核心步骤，请仔细操作：

1.  **打开 Clash for Windows 客户端**：启动您已安装的 Clash for Windows 应用程序。
2.  **进入“Profiles”（配置）选项卡**：在客户端左侧导航栏中，点击“Profiles”图标。
3.  **粘贴订阅链接**：在顶部的输入框中，粘贴您获取到的节点订阅链接。
4.  **点击“Download”（下载）**：客户端会自动从订阅链接下载并解析配置文件。下载成功后，您会在配置列表中看到新的配置文件。
5.  **选择并激活配置**：点击您刚刚下载的配置文件，使其变为绿色（激活状态）。

### 第四步：小火箭 Windows 怎么配置节点并启用代理

配置文件导入并激活后，您还需要选择合适的代理节点并开启系统代理功能。

1.  **进入“Proxies”（代理）选项卡**：在客户端左侧导航栏中，点击“Proxies”图标。
2.  **选择代理模式**：
    *   **Rule（规则模式）**：推荐使用，Clash 会根据预设规则自动判断哪些流量走代理，哪些直连。
    *   **Global（全局模式）**：所有流量都走代理，可能导致国内网站访问变慢。
    *   **Direct（直连模式）**：不使用代理。
3.  **选择节点**：在您选择的代理模式下，点击具体的代理组（如 `Proxy` 或 `国外流量`），然后选择一个延迟较低、速度较快的节点。
4.  **开启系统代理**：回到“General”（通用）选项卡，找到“System Proxy”（系统代理）选项，并将其开启。此时，您的 Windows 设备流量将通过 Clash for Windows 进行代理。

### ⚠️ 常见问题与解决方案

*   **小火箭节点失效怎么解决？**
    *   **更新订阅**：回到“Profiles”选项卡，点击配置文件旁边的刷新按钮，更新订阅。
    *   **更换节点**：在“Proxies”选项卡中尝试切换到其他节点。
    *   **检查网络**：确保您的本地网络连接正常。
    *   **联系机场**：如果问题持续，可能是机场节点本身出现问题，请联系您的服务商。
*   **Clash for Windows 订阅失败**：
    *   检查订阅链接是否正确无误。
    *   确保您的网络可以访问订阅链接。
    *   尝试更换 DNS 服务器。
    *   更新 Clash for Windows 客户端到最新版本。
