### 2. 小火箭节点为什么用不了？

当您的小火箭节点（或 Clash for Windows 中的节点）无法使用时，可能由多种原因造成：

*   **节点本身问题**：
    *   **节点失效**：节点服务器可能被封锁、维护或已停止服务。
    *   **流量耗尽/过期**：如果是付费节点，可能您的套餐流量已用完或订阅已过期。
*   **订阅链接问题**：
    *   **订阅链接失效**：免费订阅链接通常时效性短，需要频繁更新。
    *   **订阅链接错误**：复制粘贴时可能出现错误，导致无法解析。
*   **客户端配置问题**：
    *   **客户端版本过旧**：旧版本可能不支持新的协议或存在 Bug。
    *   **配置错误**：代理模式选择不当，或未正确选择节点。
*   **网络环境问题**：
    *   **本地网络故障**：您的宽带连接可能存在问题。
    *   **防火墙/安全软件**：本地防火墙或杀毒软件可能阻止了代理连接。

**解决方案**：
1.  **更新订阅**：尝试在客户端中更新订阅链接。
2.  **切换节点**：尝试连接订阅中的其他节点。
3.  **检查网络**：确保您的本地网络连接正常。
4.  **重启客户端/电脑**：有时简单的重启可以解决问题。
5.  **检查日志**：查看客户端日志，获取更详细的错误信息。

### 4. 为什么小火箭节点速度慢？

节点速度慢是用户普遍遇到的问题，原因可能包括：

*   **服务器负载高**：免费节点或热门付费节点用户过多，导致服务器拥堵，带宽分配不足。
*   **网络拥堵**：高峰时段（如晚上）国际网络线路拥堵，影响连接速度。
*   **地理位置距离**：节点服务器距离您所在的物理位置过远，数据传输延迟高。
*   **ISP 限制**：您的互联网服务提供商（ISP）可能对国际流量进行限速或QoS（服务质量）限制。
*   **节点质量差**：部分节点本身线路不佳，带宽有限，或者服务器性能不足。
*   **客户端配置不当**：代理模式选择不合理，或未开启性能优化选项。

**提升节点速度的方法**：
1.  **切换节点**：尝试连接其他节点，尤其是那些负载较低或地理位置更近的节点。
2.  **选择优质付费机场**：付费机场通常提供更优质的线路和更稳定的服务。
3.  **优化客户端设置**：检查 Clash for Windows 的代理模式和规则设置，确保其高效运行。
4.  **检查本地网络**：确保您的本地宽带速度正常，没有其他设备占用大量带宽。
